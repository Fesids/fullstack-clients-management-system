<!-- Instruções -->


### ferramentas 

* Node: 18.18.0
* postgres: 14
 
  

### clone o repositório


   ```sh
   git clone https://github.com/fesids/fullstack-clients-management-system.git
   ```

### Instalação backend

No diretório base do projeto siga os seguintes comandos
* acesse o diretório
  ```sh
  cd backend
  ```
* instale as bibliotecas
  ```sh
  npm install
  ```
* rode o projeto ( não esquece de alterar as credênciais de acesso ao banco de dados )
  ```sh
  npm run start:dev
  ```
### Instalação frontend

No diretório base do projeto siga os seguintes comandos
* acesse o diretório
  ```sh
  cd frontend
  ```
* instale as bibliotecas
  ```sh
  npm install
  ```
* rode o projeto 
  ```sh
  npm run dev
  ```
